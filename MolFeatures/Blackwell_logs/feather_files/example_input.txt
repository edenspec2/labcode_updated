Ring Vibration atoms - by order -> Pick primary atom and para to it: 
 example: 13 17
[7, 8]

Strech Vibration atoms- enter atom pairs that have a common atom: 
 example: 1,2 4,5
[12, 15]

Bending Vibration atoms - enter first atom, middle atom and last atom: 
 example: 4,7,2
[]

Indices to move center:
[5, 6, 7, 8, 10, 12]

Dipole atoms - indices for coordination transformation: 
 example: 4,5,6 - origin, y-axis, new xy plane
[5, 7, 10]

NBO values - Insert atoms to show NBO: 
 example: 1,2,3,4
[4, 2, 5]

NBO difference - Insert atoms to show NBO difference: 
 example: 1,2 3,4
[10, 14]

Sterimol atoms - Primary axis along: 
 example: 7,8
[2, 5]

Bond length - Atom pairs to calculate difference: 
 example: 1,2 4,5
[[8, 12], [6, 5]]

Bond Angle - Insert a list of atom triads/quartets for which you wish to have angles/dihedrals: 
 example: 1,3,4 5,6,7,4
[2, 5, 7]
